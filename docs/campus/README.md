## 晨跑早餐
晨跑时间：6:45 - 7:20，早餐时间：06:45 - 09:35

起床进行打卡（校园卡刷卡）晨跑，大一大二每学期有次数要求

> 早跑早超生，否则十二月会更加折磨。具体请参考 [南京邮电大学早锻炼管理规定](http://tyb.njupt.edu.cn/2020/0831/c7233a169656/page.htm)

## 上午课程
第一节课：08:00 - 08:45 && 08:50 - 09:35

第二节课：09:50 - 10:35 && 10:40 - 11:25

## 午饭午休
接下来就可以去干饭了，干完饭你可以选择学习、游戏、锻炼或者睡觉

## 下午课程
第一节课：13:45 - 14:30 && 14:35 - 15:20

第二节课：15:35 - 16:20 && 16:25 - 17:10

## 晚饭小憩
下午的课结束了，可以选择去干饭或者做点其他别的事情

## 晚自习
时间安排：18：30 - 19:15 && 19:25 - 20:10 && 20:20 - 21:05

::: tip
大一是有晚自习，到大二解放，晚自习具体安排以各学院辅导员为准
